04: Technical SEO

Construction Company
Schema Markup Implementation

Implemented JSON-LD structured data across a mid-sized construction company's website improving Google's understanding of their services and increasing eligibility for rich results.

The Problem

A regional construction company had a solid website with good content but was getting very little SERP feature visibility. Their pages weren't being picked up for any structured result types no service snippets, no local business panels, no FAQ boxes. The issue was simple: there was no structured data on the site at all.

What Was Implemented

  • LocalBusiness schema on the homepage and contact page covering name, address, phone, and service area
  • Service schema on all individual service pages (general contracting, renovation, commercial builds)
  • FAQPage schema on the FAQ section, surfacing 8 questions directly in Google results
  • BreadcrumbList schema across all pages to improve site hierarchy in search
  • Organization schema with logo, social profiles, and founding year for brand entity signals

Implementation Process

  • Audited the full site for existing structured data none found
  • Mapped out schema types appropriate for each page type
  • Wrote all JSON-LD markup and injected it via the site's WordPress theme
  • Validated all schema using Google's Rich Results Test and Schema.org validator
  • Submitted updated URLs via Google Search Console for re-crawling
  • Monitored Search Console for rich result status over the following weeks

Results

Within five weeks of implementation, the FAQ schema began triggering FAQ rich results for three service pages in Google search. The LocalBusiness panel was also updated to reflect accurate information, improving trust signals for local searches. Impressions for branded queries increased as Google confirmed the entity details.

Project Details

  • Industry: Construction
  • Schema Types: 5
  • Pages Updated: 18
  • Timeline: 2 Weeks

Schema Types

  • LocalBusiness
  • Service
  • FAQPage
  • BreadcrumbList
  • Organization

Tools

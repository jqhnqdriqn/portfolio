04: Technical SEO

Real Estate Website
Full Technical SEO Audit

Identified and resolved 180+ on-page and technical issues across a multi-location real estate website improving crawlability, reducing errors, and fixing years of accumulated technical debt.

Featured image placeholder add portfolio-audit.jpg to assets/images/

The Situation

A multi-location real estate agency came in with a site that had grown over several years without a consistent SEO strategy. They had content, they had traffic, but rankings had stagnated. The initial crawl revealed why: the site had accumulated hundreds of technical issues that were silently holding it back.

Issues Found

  • 184 broken internal and external links across property listing and location pages
  • 320+ images missing alt text or using file name as alt attribute
  • Duplicate title tags across 40+ city landing pages
  • Missing canonical tags on paginated listing pages causing index bloat
  • 14 redirect chains (3+ hops) from outdated property pages
  • XML sitemap contained 60+ URLs returning 404 or 301 status
  • Mobile tap targets below Google's minimum on key CTAs

What Was Done

  • Full crawl audit using Screaming Frog exported, categorized, and prioritized all issues by impact
  • Fixed all broken links by redirecting to active equivalents or removing from navigation
  • Rewrote alt text for all 320+ images with descriptive, keyword-aware descriptions
  • Wrote unique title tags for all duplicate pages, differentiating by location and property type
  • Added canonical tags to all paginated pages and consolidated duplicate content
  • Cleaned up the XML sitemap and resubmitted via Google Search Console
  • Documented all fixes in a shared tracking sheet with before/after status

Outcome

Within six weeks of the fixes being deployed, crawl errors in Google Search Console dropped by 78%. Previously orphaned location pages began appearing in search results for the first time. The site's overall crawl efficiency improved significantly, allowing Google to index new property listings faster.

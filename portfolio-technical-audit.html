Search Support and Technical Cleanup

Real Estate Website
Full Technical SEO Audit

A multi-location real estate website had accumulated hundreds of issues that were affecting crawlability, indexing, and overall on-page quality. I handled the audit and cleanup work needed to identify the issues, prioritize them, and resolve the highest-impact problems.

Context

A multi-location real estate agency had a site that had expanded over several years without consistent SEO upkeep. The result was a backlog of technical and on-page issues that were limiting search performance and making the site harder to maintain.

Issues Found

  • 184 broken internal and external links across property listing and location pages
  • 320+ images missing alt text or using file name as alt attribute
  • Duplicate title tags across 40+ city landing pages
  • Missing canonical tags on paginated listing pages causing index bloat
  • 14 redirect chains (3+ hops) from outdated property pages
  • XML sitemap contained 60+ URLs returning 404 or 301 status
  • Mobile tap targets below Google's minimum on key CTAs

What I Handled

  • Full crawl audit using Screaming Frog exported, categorized, and prioritized all issues by impact
  • Fixed all broken links by redirecting to active equivalents or removing from navigation
  • Rewrote alt text for all 320+ images with descriptive, keyword-aware descriptions
  • Wrote unique title tags for all duplicate pages, differentiating by location and property type
  • Added canonical tags to all paginated pages and consolidated duplicate content
  • Cleaned up the XML sitemap and resubmitted via Google Search Console
  • Documented all fixes in a shared tracking sheet with before/after status

What Changed

Within six weeks of the fixes being deployed, crawl errors in Google Search Console dropped by 78%. Previously orphaned location pages began appearing in search results for the first time, and the site became easier for search engines to crawl and index.

Why It Mattered

This work addressed the kind of technical backlog that quietly drags down visibility over time. Cleaning up the issues improved crawl efficiency, reduced avoidable errors, and gave new and existing pages a better foundation for search performance.

Project Details

  • Industry: Real Estate
  • Site Type: Multi-location
  • Issues Resolved: 180+
  • Timeline: 3 Weeks

Services Used

  • Technical SEO Audit
  • Broken Link Fixing
  • Image Alt Text
  • Canonical Tags
  • Sitemap Cleanup

Tools

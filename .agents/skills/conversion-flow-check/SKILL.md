# Conversion Flow Check

## Overview

Check whether each page leads visitors through a clean, credible path to action.

## Priority Flow

Evaluate pages in this order:

1. Hero
2. Value explanation
3. Proof
4. CTA

## Core Rules

- Make the page purpose obvious early.
- Ensure value sections answer "what is this" and "why it matters."
- Make proof appear before major action asks when possible.
- Keep CTA placement clear and timely.
- Remove friction caused by weak transitions or repeated filler sections.

## Watch For

- Heroes that do not lead naturally into value
- Value sections that arrive too late or repeat themselves
- Proof that feels buried or decorative
- CTAs that feel abrupt, weak, or too delayed
- Pages that wander instead of guiding

## Implementation Guidance

- Improve transitions before changing structure.
- Use section rhythm and emphasis to guide the eye.
- Preserve the page architecture unless a very small adjustment clearly improves flow.

## Output Expectations

When using this skill:

1. Review the page flow from top to bottom
2. Identify breaks in the conversion path
3. Refine the journey so hero, value, proof, and CTA connect clearly

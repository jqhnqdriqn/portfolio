# Readability Optimizer

## Overview

Make content easier to scan, faster to understand, and cleaner to read.

## Priorities

Work in this order:

1. Paragraph length
2. Sentence clarity
3. Scanability
4. Redundancy removal
5. CTA readability

## Core Rules

- Keep paragraphs short and purposeful.
- Remove filler, repetition, and vague phrasing.
- Prefer direct wording over soft or padded language.
- Make headings, summaries, and card copy easy to scan.
- Preserve the original meaning while improving readability.

## Avoid

- Overwriting the site's positioning
- Adding marketing fluff
- Turning clear copy into generic copy
- Making the tone robotic or over-edited

## Implementation Guidance

- Tighten before rewriting.
- Focus on comprehension speed.
- Preserve the user's core message and proof.

## Output Expectations

When using this skill:

1. Identify copy that feels dense or padded
2. Shorten and clarify it
3. Leave the page easier to read at a glance

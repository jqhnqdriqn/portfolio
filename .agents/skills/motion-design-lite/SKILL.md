# Motion Design Lite

## Overview

Add motion that supports polish, not attention-seeking effects.

## Priorities

Work in this order:

1. Hover feedback
2. Entrance and reveal motion
3. Transition timing
4. Motion consistency
5. Reduced-motion safety

## Core Rules

- Keep motion subtle and fast.
- Use motion to clarify interaction and improve feel.
- Prefer opacity, slight translate, and restrained scaling.
- Keep hover states calm and confident.
- Make similar components move in similar ways.

## Avoid

- Large movement
- Bouncy easing
- Flashy animation
- Multiple competing motion styles
- Motion that slows the site or distracts from content

## Implementation Guidance

- Favor CSS transitions over heavier JS when possible.
- Keep durations restrained and consistent.
- Respect `prefers-reduced-motion`.
- Apply motion where it improves perceived quality, not everywhere.

## Output Expectations

When using this skill:

1. Find areas that benefit from subtle motion
2. Apply clean hover and reveal behavior
3. Keep the site fast, calm, and professional

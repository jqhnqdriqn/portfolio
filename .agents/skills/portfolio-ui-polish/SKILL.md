# Portfolio UI Polish

## Overview

Polish this portfolio like a premium client-facing site. Improve visual quality, consistency, and perceived professionalism without changing the core positioning or rewriting the site strategy.

## Polish Priorities

Work in this order:

1. Visual hierarchy
2. Section spacing and rhythm
3. Card and button polish
4. Typography clarity
5. Subtle motion and hover states
6. Small trust-building details

## What to Change

### Spacing and Layout Rhythm

- Tighten sections that feel too loose or repetitive.
- Add breathing room where content feels crowded.
- Make transitions between sections feel deliberate and smooth.
- Preserve the current layout unless a small structural adjustment clearly improves polish.

### Typography

- Improve heading emphasis, paragraph readability, and label hierarchy.
- Keep text blocks scannable and visually balanced.
- Prefer fewer, clearer emphasis moments over decorative styling everywhere.

### Cards and Repeated Components

- Make repeated cards feel aligned in density, padding, and visual weight.
- Standardize image, title, body copy, and CTA relationships.
- Keep hover states subtle and confident.

### Buttons and CTAs

- Keep CTA styling clear and consistent.
- Make primary proof and action paths visually obvious.
- Avoid introducing extra button variations unless absolutely necessary.

### Motion

- Use light, purposeful motion only.
- Prefer:
  - subtle lift on hover
  - smooth opacity/translate reveals
  - restrained transition timing
- Avoid:
  - flashy animation
  - large movement
  - multiple competing motion styles

## Guardrails

- Do not redesign the site.
- Do not change the core content strategy unless the user explicitly asks.
- Do not add flashy visual gimmicks.
- Do not introduce trend-heavy effects that reduce trust.
- Keep the result feeling premium, clean, and client-facing.

## Implementation Style

- Favor small, high-leverage refinements.
- Reuse the existing design system where possible.
- Keep edits cohesive across pages instead of polishing one section in isolation.
- If adding CSS, prefer a minimal set of reusable adjustments over scattered one-off rules.

## Output Expectations

When applying this skill:

1. Identify the highest-impact polish opportunities
2. Implement the smallest changes that create visible improvement
3. Verify consistency across the affected pages
4. Summarize what changed and why it improves the premium feel

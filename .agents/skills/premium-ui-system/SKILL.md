# Premium UI System

## Overview

Apply a premium design standard to this portfolio. Keep the work restrained, intentional, and confident.

## Priorities

Work in this order:

1. Visual hierarchy
2. Spacing and balance
3. Typography quality
4. Section rhythm
5. Premium restraint

## Core Rules

- Make the most important message visually dominant.
- Use spacing to create rhythm, not empty space.
- Keep typography elegant and easy to scan.
- Favor fewer, stronger visual accents over decorative styling everywhere.
- Make sections feel curated, not stacked from a template.

## Avoid

- Generic card-grid sameness
- Overuse of labels, outlines, or decorative accents
- Layouts that feel AI-generated or interchangeable
- Excess visual noise
- Trend-heavy styling that reduces trust

## Implementation Guidance

- Preserve the existing information architecture unless a very small layout shift clearly improves polish.
- Reuse the existing design language instead of introducing a new one.
- Prefer small, high-leverage refinements that improve perceived quality immediately.

## Output Expectations

When using this skill:

1. Identify where the design feels flat or generic
2. Apply cleaner hierarchy and spacing
3. Keep the result minimal, polished, and high-end
